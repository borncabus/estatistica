Intervalo de Confiança para o Quantil Populacional

Exemplo em R

Sobre o Intervalo de Confiança para o Quantil Populacional

O intervalo de confiança para um quantil populacional é uma ferramenta estatística que permite estimar com um certo nível de confiança o valor de um quantil (por exemplo, o quartil ou percentil) de uma população, a partir de uma amostra de dados.

Em R, isso pode ser calculado usando funções como quantile() para estimar o quantil da amostra e qnorm() ou qt() para determinar os limites do intervalo de confiança, assumindo uma distribuição normal ou t de Student, respectivamente.

Exemplo em R


# Dados de exemplo
dados <- c(23, 25, 27, 30, 32, 35, 36, 38, 40, 45)

# Calculando o intervalo de confiança para o quartil 0.25
quantil <- quantile(dados, probs = 0.25)
n <- length(dados)
alpha <- 0.05

# Calculando os limites do intervalo de confiança
limite_inferior <- quantil + qnorm(alpha/2) * sqrt(quantil * (1 - quantil) / n)
limite_superior <- quantil + qnorm(1 - alpha/2) * sqrt(quantil * (1 - quantil) / n)

# Exibindo o resultado
cat("Intervalo de Confiança para o Quartil 0.25: (", limite_inferior, ",", limite_superior, ")\n")

\[ \bar{X} \pm z_{\frac{\alpha}{2}} \cdot \frac{\sigma}{\sqrt{n}} \]

\[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x - \mu)^2}{2\sigma^2}} \]
