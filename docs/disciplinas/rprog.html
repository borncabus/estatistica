Elementos de Programação para Estatística

CM210 | 60 Horas

Sobre a Matéria

Elementos de Programação para Estatística é uma disciplina que introduz os fundamentos da programação aplicada à estatística. Os alunos aprenderão a utilizar linguagens de programação e software estatísticos para manipulação de dados, execução de análises e visualização de resultados. A disciplina abrange desde conceitos básicos de programação até técnicas avançadas de análise de dados.

Os tópicos incluem introdução às linguagens de programação (como R e Python), manipulação de dados, funções, loops, estruturas condicionais, importação e exportação de dados, visualização de dados e implementação de métodos estatísticos. A disciplina é essencial para aqueles que desejam aplicar técnicas estatísticas de forma eficiente e automatizada.
